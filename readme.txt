== Description ==

Simple FAQ gives you ability to create very simple FAQ on your site (questions and answers). You can add question and answer thru the Wordpress panel.

== Installation ==

1. Unzip the package and upload the Simple FAQ directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Place `[display_faq]` in your page
